== Description ==

A simple plugin that allows you to display your shots feed from Dribbble in a sidebar area of your website.

[Live Demo](https://outlane.co)

Don't forget to get your Dribbble Api Key [here](https://dribbble.com/account/applications/new). Fill in the form and copy your Client Access Token, then paste it in the widget settings.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'Recent Shots Widget'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard
5. Go to 'Appearance' => 'Widgets' and select 'Recent Shots Widget'
6. Fill in your Dribbble Username and your Dribbble API KEY (which you can get for free [here](https://dribbble.com/account/applications/new). Register your application and copy Client Access Token)

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `recent-shots-widget.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard
6. Go to 'Appearance' => 'Widgets' and select 'Recent Shots Widget'
7. Fill in your Dribbble Username and your Dribbble API KEY (which you can get for free [here](https://dribbble.com/account/applications/new). Register your application and copy Client Access Token)
